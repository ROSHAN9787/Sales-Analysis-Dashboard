# Sales Dashboard Power BI Project

Interactive sales dashboard created using Microsoft Power BI for analyzing sales, profit, quantity, and business insights.

## Features
- Sales Analysis
- Profit Tracking
- KPI Cards
- Interactive Charts
- Regional Analysis
- Map Visualization

## Tools Used
- Power BI
- Excel / CSV
- DAX
- Data Visualization

## Files Included
- Power BI Dashboard (.pbix)
- Dataset
- Dashboard Screenshots

## How to Use
Download the `.pbix` file and open it in Microsoft Power BI Desktop.
